# Dashboard de Vendas Xbox

## Descrição do Projeto
Este projeto consiste na criação de um dashboard de vendas do Xbox no Excel, que transforma dados brutos de assinaturas em informações visuais para análise de desempenho.

## Dados Utilizados
- Subscriber ID
- Nome do Cliente
- Tipo de Assinatura (Monthly, Quarterly, Annual)
- EA Play Season Pass
- Minecraft Season Pass
- Valor de Cupons aplicados
- Total Value

## Funcionalidades
- Resumo de vendas total e média por assinatura
- Gráficos de assinatura por tipo
- Gráficos de receita por tipo de assinatura
- Indicadores de assinaturas com EA Play e Minecraft
- Tabela com top 5 clientes por receita
- Análise de cupons aplicados

## Instruções para Reprodução
1. Abrir o arquivo Excel `Dashboard_Xbox.xlsx`.
2. Navegar até a aba `Dashboard` para visualizar os gráficos e resumos.
3. As tabelas e gráficos são baseados em Tabelas Dinâmicas, então podem ser atualizados automaticamente ao alterar os dados na aba `Dados`.
